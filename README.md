**GNES**  
GNES is a cross-platform NES Emulator made to gain a greater understanding of how Emulators and the NES work. There are future plans for unique ideas to build upon this emulator in the future, but accuracy and playability is the current focus to gain a strong foundation.  

**NES Hardware Documentation Sources**  
The links below are the sole sources of information about the NES that were used to implement this NES Emulator.  
NESDEV: https://www.nesdev.org/ and https://www.nesdev.org/NESDoc.pdf  
6502.org: http://www.6502.org/  
Everynes: https://d1.amobbs.com/bbs_upload782111/files_28/ourdev_551332.pdf  

**Supported Platforms**  
Arch Linux and Windows 10 on the x86-64 platform.
